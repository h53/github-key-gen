# github-keygen

```
bash -c "$(curl -fsSL https://github-keygen.pages.dev/run.sh)" [prefix] [suffix]
```

---

### config git user/email
```
git config user.email "github-keygen `uname`-`arch` [`date`]"
git config user.name "`whoami`"
```

### commit changes
```
git add .
git commit -m "update"
git push origin main
```
